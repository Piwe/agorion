How It Works

A simple protocol loop that turns static integrations into dynamic service discovery.

1

Publish manifest

Providers add /.well-known/agent-services.json to declare callable endpoints and terms.

2

Agents discover

Discovery registries index manifests so agents can query by capability, chain, and budget.

3

Auto-negotiate & pay

Agents pick highest-value services using price and reputation signals, then settle via x402 when needed.

For API Providers

Register your endpoints. Get discovered by thousands of AI agents. Get paid via x402.

ASDP lets existing APIs become agent-callable without rebuilding your stack. Publish once, then update services as pricing and capabilities evolve.

For Agent Builders

Your agents find the best data sources automatically. Budget-aware. Quality-scored.

Replace static API lists with runtime discovery. Route to providers by capability, latency, uptime, and price constraints set by your policy engine.
